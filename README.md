# peer-review-cli

Generate peer review email from a markdown template and Visual Studio Team Services work item number.

## Install

`npm install @trodi/peer-review-cli`

## Usage

`peer-review --help`

## Build

* `npm i`
* `npm run build`

## Publish

* `npm version <major|minor|patch>`
* `npm publish`
